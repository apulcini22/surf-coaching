Auto-Populated Shopify Data (Page Load)

These Shopify global objects are set on window automatically when the page loads, simulating a completed Shopify checkout / thank-you page.

window.Shopify Immediate Shop config, locale, currency ✓ Set
ShopifyAnalytics.meta Immediate Page type & product metadata ✓ Set
Shopify.checkout Immediate Completed order / checkout object ✓ Set
Shopify.Checkout.step Immediate thank_you step identifier ✓ Set
dataLayer purchase +3 seconds GA4-style purchase derived from Shopify order Pending
